========
Elements
========

Elements help structure and present content effectively, they range from text-based components like
:ref:`titles <website/web_design/elements/titles>` and highlighted text to interactive ones such as
:ref:`buttons <website/web_design/elements/buttons>` and links. Visual elements like images, icons,
and videos can be added to better represent content and organize information.

To modify, all elements on website:

#. Click on :guilabel:`edit`.
#. Select the section where you want to add an element.
#. Once changes are done, click on :guilabel:`Save`.

.. _website/web_design/elements/titles:

Titles
======

Titles define headings, they organize content on a website into different levels for clarity and
structure. To insert a title type `/title`, choose among three different headings are available:
:guilabel:`Heading 1`, :guilabel:`Heading 2` and :guilabel:`Heading 3` and type the text.

.. _website/web_design/elements/buttons:

Buttons
=======

Buttons are interactive elements that allow website vistors to trigger actions. They improve
navigation and engagement, they can be a link to another page, a form, or a pop-up.
To insert a button type `/button`:

#. To display a label on the button type :guilabel:`Link Label`.
#. Add the :guilabel:`URL or Email`.
#. For visual elements, :guilabel:`Style`, :guilabel:`Size` and :guilabel:`Layout` can be set to
   define the button's appearance.
#. Click :guilabel:`Apply` to save the changes.

.. tip::
   To modify an existing button click on it to open the website editor and access editing option
   under the :guilabel:`Customize` tab in the :guilabel:`Inline text` section.

Images
======

Images add visual content, improving presentation and clarity. To insert an image type `/image`,
click on :guilabel:`Upload an image` to choose a file from your local images. Click on the image to
open the website editor, here they can be customized, :guilabel:`replaced`, :guilabel:`Description`
and :guilabel:`Tooltip` options are available to add alt tags and title tag.
