========
Elements
========

Elements help structure and present content effectively, they range from text-based components like
:ref:`titles <website/web_design/elements/titles>` and highlighted text to interactive ones such as
:ref:`buttons <website/web_design/elements/buttons>` and :ref:`links
<website/web_design/elements/links>`.Visual elements like :ref:`images
<website/web_design/elements/images>`, :ref:`icons <website/web_design/elements/icons>`, and
:ref:`videos <website/web_design/elements/videos>` can be added to better represent content and
organize information.

To modify, all elements on website:

#. Click on :guilabel:`edit`.
#. Select the section where you want to add an element.
#. Once changes are done, click on :guilabel:`Save`.

.. _website/web_design/elements/titles:

Titles
======

Titles define headings, they organize content on a website into different levels for clarity and
structure. To insert a title type `/title`, choose among three different headings are available:
:guilabel:`Heading 1`, :guilabel:`Heading 2` and :guilabel:`Heading 3` and type the text.

.. _website/web_design/elements/buttons:

Buttons
=======

Buttons are interactive elements that allow website vistors to trigger actions. They improve
navigation and engagement, they can be a link to another page, a form, or a pop-up.
To insert a button type `/button`:

#. To display a label on the button type :guilabel:`Link Label`.
#. Add the :guilabel:`URL or Email`.
#. For visual elements, :guilabel:`Style`, :guilabel:`Size` and :guilabel:`Layout` can be set to
   define the button's appearance.
#. Click :guilabel:`Apply` to save changes.

.. tip::
   To modify an existing button click on it to open the website editor and access editing option
   under the :guilabel:`Customize` tab in the :guilabel:`Inline text` section.

.. _website/web_design/elements/images:

Images
======

Images add visual content, improving presentation and clarity. To insert an image type `/image`,
click on :guilabel:`Upload an image` to choose a file from your local images. Click on the image to
open the website editor, where you can customize it in the :guilabel:`Image` section by replacing
it, adding a :guilabel:`shape` to it, and configuring options like the :guilabel:`alt tag` and
:guilabel:`title tag`.

.. _website/web_design/elements/videos:

Videos
======

Videos are dynamics elements that improve website engagement through visual content. To add a video
type `/video`, insert the URL, and modify it with options like :guilabel:`Autoplay` and
:guilabel:`Loop`.

.. _website/web_design/elements/icons:

Icons
=====

Icons are visual elements that enhance website design and usability, they help convey information
quickly and improve navigation. To insert an icon type `/image` and open the :guilabel:`Icons` tab.
To modify an icon, click on it and on the website editor, customize it with options like
:guilabel:`Color`, :guilabel:`Size`, :ref:`Animations <website/web_design/elements/animations>`
:guilabel:`Shape`.

.. _website/web_design/elements/links:

Links
=====

Links are essential elements that connect different pages and resources, guiding visitors and
improving navigation. To add a link type `/link`, a pop up opens where :guilabel:`Label`,
:guilabel:`URL or Email` and :guilabel:`Style` can be modified.

.. _website/web_design/elements/lists:

Lists
=====

Lists help organize content clearly, making information easier to read and improving web pages'
structures. Type `/list`, here three different types of lists are available:
:guilabel:`Bulleted lists`, :guilabel:`Numbered lists` and :guilabel:`Checklists`.

.. _website/web_design/elements/animations:

Animations
==========

Animations add movement to website elements, enhancing visual aspects and improving visitors
engagement. There are two different types of animations: :guilabel:`on scroll`,
:guilabel:`on appearance`. To add an animation:

#. Click on an element.
#. On the website editor click on the :guilabel:`Animation` field.
#. Choose the type of animation.

According to the animation, different options are available. For :guilabel:`animations on scroll`
it is possible to:

 - Choose :guilabel:`in` to add the animation when the element enters the screen
   and :guilabel:`out` to add it when it leaves the screen.
 - Choose among the different effects.
 - Choose the :guilabel:`Direction` of different effects.
 - Adapt the :guilabel:`Intensity` of the effect.
 - Define the :guilabel:`Scroll Zone`, where the first value represents the % of the screen when the
   effect starts and the second value represents its end.

For :guilabel:`animations on appearence` it is possible to:

 - Choose among different effects.
 - Choose the :guilabel:`Direction` of the effects.
 - Pick a :guilabel:`Trigger` option to either have the animation happen the
  :guilabel:`First Time only` or :guilabel:`Every Time`.
 - Adapt the :guilabel:`Intensity` of the effect.
 - Define a :guilabel:`Start After` if you want the animation to be triggered after an amount of
   seconds.
 - Choose the length of the :guilabel:`Duration` for the animation.
