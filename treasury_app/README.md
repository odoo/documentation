# نظام إدارة الخزينة والعهدة
## Treasury & Expense Management System

برنامج ويب مستقل لإدارة الخزينة والعهدة ومطالبات السداد.

### المتطلبات
- Python 3.10+

### التثبيت والتشغيل

```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. تشغيل البرنامج
python run.py

# 3. افتح المتصفح
# http://localhost:5000
```
